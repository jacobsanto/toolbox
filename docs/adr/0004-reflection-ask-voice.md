# ADR 0004 — Reflection, «Ask the OS» & Voice

- **Κατάσταση:** Αποδεκτό
- **Ημερομηνία:** 2026-06-09

## Πλαίσιο

Phase 5: το OS αποκτά αναστοχασμό (η δραστηριότητα γίνεται ημερολόγιο),
ενιαίο σημείο ερωτήσεων στα Ελληνικά, και φωνή. Στόχος: «μιλάω σε ένα
πράγμα, αυτό ξέρει τα πάντα, απαντάει στα Ελληνικά».

## Απόφαση

| Θέμα | Επιλογή | Αιτιολόγηση |
|---|---|---|
| Reflection | **Deterministic recap από τη SQLite + προαιρετικό AI αφήγημα** | Τα νούμερα (εκτελέσεις, σφάλματα, scopes, inbox) βγαίνουν πάντα, δωρεάν, από τη βάση. Το `--ai` προσθέτει 2-3 προτάσεις αφήγησης μέσω πραγματικού LLM run (καταγεγραμμένο, με κόστος). Γράφεται στο `Journal/<date>.md` και `Journal/Weekly/<year>-W<ww>.md` με wikilinks προς τα session notes — το Obsidian γίνεται ο «καθρέφτης της ημέρας». |
| Ask the OS | **Ενιαία αναζήτηση (notes+runs+inbox) → προαιρετική σύνθεση LLM με citations** | Η αναζήτηση είναι το ίδιο accent-insensitive tokenize του memory layer — μία υλοποίηση παντού. Η σύνθεση περνά από τον runner (dogfooding) με αρίθμηση πηγών [n]· ο LLM εντολοδοτείται να απαντά ΜΟΝΟ από τις πηγές και να λέει καθαρά πότε δεν αρκούν (επιβεβαιώθηκε πειραματικά). Στο API η σύνθεση είναι opt-in (`synthesize_answer=true`) γιατί είναι αργή/κοστίζει. |
| STT | **whisper CLI (τοπικό)** μέσω subprocess | Ελληνικά, δωρεάν, offline. Wrapper με ανίχνευση + οδηγία εγκατάστασης. |
| TTS | **edge-tts CLI** (default φωνή el-GR-NestorasNeural) | Δωρεάν ελληνικές φωνές. Θέλει δίκτυο (Microsoft endpoint) — σε sandboxed περιβάλλον αποτυγχάνει με καθαρό μήνυμα, στο Mac δουλεύει. Το ElevenLabs μπαίνει αργότερα ως premium εναλλακτική στο ίδιο interface. |
| Proactive | `mission-control brief [--speak]` | Χρονο-ευαίσθητος χαιρετισμός + running/σημερινά runs/σφάλματα/inbox count. Σχεδιασμένο για cron/launchd («κάθε πρωί στις 9»). |

## Συνέπειες

- **Θετικές:** το ημερολόγιο γράφεται μόνο του· μία ερώτηση ψάχνει παντού· το voice layer είναι λεπτό wrapper πάνω από τα ίδια δεδομένα — όχι ξεχωριστό σύστημα.
- **Αρνητικές / αποδεκτά trade-offs:**
  - Η σύνθεση απαντήσεων κοστίζει ένα LLM run ανά ερώτηση — γι' αυτό είναι opt-in στο API και ορατή ως καταγεγραμμένο run.
  - Το edge-tts εξαρτάται από εξωτερικό endpoint· το whisper θέλει τοπική εγκατάσταση (μεγάλο dependency) — και τα δύο εκτός των core deps του project.
  - Το recap βασίζεται μόνο σε ό,τι πέρασε από το Mission Control — ό,τι γίνεται εκτός OS δεν καταγράφεται (by design).
